# IPUT学生団体ホームページ

## Directory Configuration
```
.
├── assets/
    ├── style-footer.css
    ├── style-header.css
    ├── style-index.css
    ├── style-login.css
    ├── style-news.css
├── images/
├── js/
    ├── author.js
├── .htaccess - サイト設定ファイル
├── screenshot.png - テーマのヘッダー画像
├── style.css - テーマのCSS
├── author.php
├── post.php - 記事投稿プログラム
```

## 固定ページ
- front-page
- page-login
- page-activity
- page-news
- page-faq
> **Note**\
> Wordpressのホームページは`設定 > 表示設定 > ホームページの表示 > 固定ページ`からドロップダウンで選択して変更できる
